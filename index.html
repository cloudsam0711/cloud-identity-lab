Active Directory Lab on Azure

Author: Sam Glasper

This lab simulates a real-world Active Directory Domain Services (AD DS) environment using a Windows Server 2022 VM hosted in Microsoft Azure.

Lab Overview

  • Created and configured a VM in Azure with Windows Server 2022
  • Installed AD DS and DNS roles
  • Promoted server to domain controller for glasperlabs.local
  • Created users, OUs, and security groups
  • Configured Group Policy and DNS records

Technologies Used

  • Microsoft Azure
  • Windows Server 2022
  • Active Directory Domain Services
  • Group Policy Management
  • DNS Server
